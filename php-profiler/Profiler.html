 Method setTrivialThreshold (line 399)
 
 

void setTrivialThreshold(
float
$threshold)


	

Set the Percentile Boundary Threshold


This is used to set the percentile boundary for when a node is considered trivial or not.  By default, .75 is used.  This translates to the fastest 25% of nodes being regarded "trivial".  This is a sliding scale, so you will always see some output, regardless of how fast your application runs.

		
Parameters

	
    
			
  • float $threshold: the threshold to use as the percentile boundary
    •
